# gulp-start-boilerplate

## Структура:
В папке ./app - файлы дев-версии
В папке ./build - файлы прод-версии, обработанные сборщиком Gulp

## Требования:
> **Node.js**

## Подготовка:
1. `npm init`
2. `npm install`

## Запуск сборщика дев-версии (watcher + browserSync):
- `npm run dev`

## Запуск сборщика прод-версии (builder):
- `npm build:prod`
